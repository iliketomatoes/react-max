# React demo app

Front end application built with:

    * React 16
    * Webpack 4
    * Typescript 3
    * Material UI 3


## Requirements

### Node >=8.9.4
### NPM ^6.1.0

## Installation

```bash
npm install
```

## Usage

### Development

`npm run dev`

### Lint

`npm run lint`

### Build for production

`npm run build`
